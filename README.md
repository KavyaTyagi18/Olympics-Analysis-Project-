# 🏅 Olympic Analysis Web Application

An interactive **Olympic Data Analysis Dashboard** built using **Python, Streamlit, Pandas, Plotly, Matplotlib and Seaborn**.

The application allows users to explore Olympic Games statistics, medal tallies, athlete performance, country-wise analysis and various insightful visualizations generated from historical Olympic datasets.

---

# 🚀 Live Features

✅ Medal Tally Analysis

✅ Overall Olympic Statistics

✅ Country-wise Performance Analysis

✅ Athlete-wise Analysis

✅ Interactive Charts & Graphs

✅ Heatmaps and Data Visualization

✅ Age Distribution Analysis

✅ Height vs Weight Analysis

✅ Gender Participation Trends

# 📊 Dataset

The project uses the **Olympic History Dataset** obtained from **Kaggle**.

The dataset contains information regarding:

- Olympic Athletes
- Events
- Sports
- Countries
- Medal Winners
- Gender Participation
- Height & Weight
- Olympic Editions

# ▶️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/OlympicAnalysisWebApp.git
```

Move to project folder

```bash
cd OlympicAnalysisWebApp
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
